🔹 Autonoleggio Web App

Sistema completo di gestione noleggio auto con:
	•	Ruoli: Admin e Cliente
	•	CRUD veicoli
	•	Prenotazioni con UUID
	•	Download fattura

Applicazione chat leggera e multipiattaforma:

	•	Invio messaggi e immagini ottimizzate
	•	Autenticazione anonima e gestione dispositivi
